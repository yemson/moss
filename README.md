# Moss

Moss는 흩어진 구독을 한곳에 모아 보고, 다음 결제를 놓치지 않도록 정리해주는 iOS 구독 관리 앱이다.  
복잡한 가계부보다는 `지금 어떤 구독을 쓰고 있는지`, `이번 달에 얼마나 나갈지`, `어디에 많이 쓰고 있는지`를 빠르게 파악하는 데 집중한다.

## 한눈에 보는 앱

- 이용 중인 구독을 한 화면에서 모아본다.
- 이번 달 남은 결제와 이번 달 전체 결제 규모를 바로 확인한다.
- 자주 쓰는 서비스는 템플릿으로 빠르게 추가하고, 필요하면 직접 입력한다.
- 구독별 결제 흐름과 최근 결제 내역을 확인한다.
- 전체 통계 화면에서 최근 6개월 흐름과 카테고리별 지출 패턴을 본다.
- 결제 하루 전 알림으로 놓치기 쉬운 구독을 챙긴다.

## 어떤 문제를 해결하나

구독은 하나하나 금액이 크지 않아도 쌓이면 체감보다 빠르게 커진다.  
Moss는 구독을 등록하는 순간부터 결제일과 결제 로그를 기준으로 현재 상태를 정리하고, “이번 달에 얼마나 남았는지”와 “지금까지 얼마나 썼는지”를 자연스럽게 보여준다.

## 핵심 경험

### 1. 홈에서 바로 파악

- 이번 달 남은 예정 금액과 이번 달 전체 결제 금액을 요약 카드로 보여준다.
- 구독 목록은 결제일 순, 결제액 순으로 정렬할 수 있다.
- 카테고리 필터로 원하는 구독만 빠르게 좁혀 볼 수 있다.
- 각 구독 카드는 카테고리, 결제 주기, 다음 결제 시점을 한 줄에서 간결하게 보여준다.
- 카드 스와이프로 수정과 삭제를 바로 수행할 수 있다.
- 구독이 없을 때는 홈 전체를 비우고 추가 액션에만 집중한 빈 상태를 제공한다.

### 2. 단계형 구독 추가

- 첫 단계에서 자주 쓰는 서비스 템플릿을 고르거나 `직접 입력`으로 시작한다.
- 서비스 이름, 카테고리, 결제액, 결제일, 결제 주기, 알림 여부, 메모를 순서대로 입력한다.
- KRW만 지원하며 결제액은 정수 입력 기준으로 단순화했다.
- 입력 흐름은 모바일에서 빠르게 끝낼 수 있도록 단계형으로 나눴다.

### 3. 구독 상세에서 흐름 확인

- 구독의 기본 정보와 현재 상태를 바로 확인할 수 있다.
- 누적 결제액, 결제 횟수, 올해 결제액, 앞으로 12개월 예정 금액을 요약으로 보여준다.
- 최근 6개월 결제 흐름 차트로 이 구독에 얼마나 썼는지 본다.
- 최근 결제 내역을 최신 순으로 확인할 수 있다.

### 4. 통계로 전체 패턴 이해

- 이용 중인 구독 수, 이번 달 남은 결제, 이번 달 전체 결제, 지금까지 결제한 금액을 요약한다.
- 앞으로 12개월 예정 금액으로 장기적인 지출 규모를 미리 본다.
- 최근 6개월 결제 흐름 차트로 월별 지출 변화를 확인한다.
- 카테고리별 지출로 어디에 가장 많이 쓰는지 파악한다.

### 5. 설정과 알림

- 앱 전체 알림을 켜고 끌 수 있다.
- 시스템 알림 권한이 꺼져 있으면 설정으로 바로 이동할 수 있다.
- 화면 모드는 앱 안에서 바로 바꿀 수 있다.
- 개발 빌드에서는 햅틱 테스트, 알림 테스트 같은 내부 검증 화면을 숨겨서 제공한다.

## 동작 방식

- 데이터의 기준은 로컬 SQLite다.
- 구독 정보를 바탕으로 결제 로그를 자동으로 생성하고, 요약 카드와 통계는 이 로그를 기준으로 계산한다.
- 과거 결제는 `paid`, 앞으로의 결제는 `scheduled`로 관리한다.
- 구독 정보를 수정하면 과거 기록은 유지하고, 앞으로의 예정 결제만 최신 정보로 다시 반영한다.
- 알림은 앱 전체 설정과 개별 구독 설정을 함께 반영해 예약한다.

## 제품 원칙

- 복잡한 재무 관리 앱보다 빠르게 이해되는 구독 관리 경험을 지향한다.
- iOS에서 자연스럽고 부담 없는 조작을 우선한다.
- 네트워크 의존성을 최소화하고, 로컬 데이터 중심으로 안정적으로 동작하게 만든다.
- 통계는 자세함보다 “한눈에 파악되는 흐름”을 우선한다.

## 기술 요약

- Expo Router 기반 iOS 앱
- 로컬 저장소: Expo SQLite
- UI: HeroUI Native, Uniwind
- 차트: `react-native-svg` 기반 커스텀 렌더링
- 알림: Expo Notifications
- 광고: AdMob 배너

## 개발 시작

```bash
npm install
npm run ios
```

Metro가 필요할 때는 아래 명령을 사용한다.

```bash
npm run start
```

린트는 아래 명령으로 실행한다.

```bash
npm run lint
```

## 개발 메모

- 이 프로젝트는 iOS 전용이다.
- Expo Go가 아니라 iOS dev build 기준으로 작업한다.
- 주요 코드는 아래 구조를 따른다.
  - `src/app`: 라우트와 화면 조합
  - `src/components`: 재사용 UI
  - `src/lib`: 저장, 통계, 포맷, 알림 같은 비 UI 로직
